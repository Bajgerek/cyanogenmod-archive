It’s been a few weeks of work, and a great effort from everyone involved! I’d like to present the first CM6 release candidates for the Nexus One, Dream, and Magic phones! Versions for other devices are on the way.

Check out the CHANGELOG for a full list of features and contributors.

  • http://www.BostonGreekGeek.com harpocrates23

    @ CJ:

    CM6 does not have an auto apps2sd. It does have an option in settings->cyanogen settings to allow moving of applications to the sd card and to allow new apps to go direct to external storage. You have to go move all the apps from the phone memory to the sd card right now through applications->manage applications and that will free up phone memory and speed up your phone. However if you choose to have apps installed direct to external memory on the G1 you might encounter installation failures. If this happens install to phone memory and move…its a pain but its a good enough work around.

    A later update of CM6 might have auto apps2sd but we have to wait.
