Tutorial videos wanted, can you help?

Guys and Girls.

With the recent registration of our own YouTube channel, I’m looking for people who are able to create tutorial videos and other videos that can help users with the CyanogenMod.

Videos that are required :

1) Rooting.
2) Installing CynogenMod Recovery Image.
3) Flashing a new image manually.
4) Flashing a new image via the updater.
5) Flashing a theme.

More videos will follow but these are the main ones for now.

Wannabe producers would talk the procedure through (this can be computer voiced) and explain to viewers how to complete the above.

The videos will be set in a blank template (somewhere clean and clear of rubbish) and shall be presented in a professional manner.

If you are able to do this, please send a message to @Cyanogenmod on Twitter.
