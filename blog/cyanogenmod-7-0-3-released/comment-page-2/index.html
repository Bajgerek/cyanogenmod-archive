CyanogenMod 7.0.3 Released

CyanogenMod 7.0.3 is now available for download. This update contains a bug fix for our update notification system, as well as an important security fix. It is recommended that all users running a version of CyanogenMod prior to 7.0.3 update to this release. Â You should not install this update if you are already running a recent nightly build, only if you are coming from 7.0, 7.0.2, or 7.0.2.1.

As always, you can grab the latest version over at the mirror network, powered by TDRevolution. This update is also available on Rom Manager.

Please note that this release is still based on Android 2.3.3. Â CyanogenMod 7.1 will include Android 2.3.4 features and updates.
