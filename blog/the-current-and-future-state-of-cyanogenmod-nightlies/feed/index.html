For the user this is a blackbox, you have root access to the VM and the VM calls a server every X Minutes and asks for build-jobs. The build jobs which are offered will loop through all devices. The VM compiles and sents the result back via SSH/FTP etc. You wait until there are N identical builds e.g. same hash, then publish this build as nightly and delete the according open build job. 
So you don’t have to care if one special machine is online and I’m quite sure there will be hundrets of user who support this project by providing cpu-usage of their systems. If the build time is arround 45minutes on a octalcore with 24GB Rams, I would bet that you have builds of 50+ devices in less than two hours including down- and upload of the files.
